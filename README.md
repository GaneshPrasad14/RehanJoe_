# Rehan Joe | Fractional CRO & Revenue Growth Advisor

This project is the portfolio and advisory website for Rehan Joe. It is built as part of a **MERN Stack** (MongoDB, Express, React, Node.js) architecture, though it currently operates as a frontend-only React.js site without an active database or backend.

## Technologies Used
- React 19 (Frontend)
- Node.js & Express (Available in dependencies for future backend)
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Build for production:
   `npm run build`
